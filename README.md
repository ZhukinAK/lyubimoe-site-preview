# Место для двоих

Приватный сайт для двоих: игры, галерея, полезные ссылки и общая лента воспоминаний.

## Как открыть

Откройте файл `index.html` в браузере. Сайт работает без сборки и сервера.

## Что уже есть

- Главная страница с навигацией.
- Раздел игр с рабочей Виселицей: слова из общего набора и режим, где можно загадать слово с подсказкой друг для друга.
- Будущие игровые режимы: Балда и вопросы друг другу.
- Галерея с добавлением картинки и подписи.
- Лента воспоминаний с добавлением заметок.
- Сохранение данных в `localStorage` браузера.
- Адаптивная верстка под телефон и десктоп: прокручиваемая мобильная навигация, крупные зоны нажатия, компактные формы и стабильная сетка букв в игре.

## Текущая архитектура

Сейчас данные сохраняются только в браузере. В следующих версиях:

- интерфейс переедет в `Next.js`;
- авторизация и база будут в `Supabase`;
- картинки будут храниться в `Supabase Storage`;
- игры получат realtime-синхронизацию для двух пользователей;
- сайт можно будет развернуть на `Vercel` и привязать к домену.

## Ближайшие шаги

1. Наполнить сайт реальными ссылками, заметками и галереей.
2. Проверить основные сценарии на телефоне и десктопе.
3. Перенести сайт в `Next.js`.
4. Подключить `Supabase`.
5. Сделать приватный вход для двух пользователей.
6. Переделать Виселицу из локальной игры в игру с комнатой и realtime.
