# Место для двоих

Локальный прототип приватного сайта для двоих: игры, галерея и общая лента воспоминаний.

## Как открыть

Откройте файл `index.html` в браузере. Прототип работает без сборки и сервера.

## Что уже есть

- Главная страница с навигацией.
- Раздел игр с рабочей Виселицей: демо-слова и режим, где можно загадать слово с подсказкой друг для друга.
- Заготовки будущих игр: Балда и вопросы друг другу.
- Галерея с добавлением картинки и подписи.
- Лента воспоминаний с добавлением заметок.
- Сохранение демо-данных в `localStorage` браузера.
- Адаптивная верстка под телефон и десктоп: прокручиваемая мобильная навигация, крупные зоны нажатия, компактные формы и стабильная сетка букв в игре.

## Что это имитирует

Сейчас данные сохраняются только в браузере. В полноценной версии:

- интерфейс переедет в `Next.js`;
- авторизация и база будут в `Supabase`;
- картинки будут храниться в `Supabase Storage`;
- игры получат realtime-синхронизацию для двух пользователей;
- сайт можно будет развернуть на `Vercel` и привязать к домену.

## Ближайшие шаги

1. Уточнить общий стиль: более нежный, более минималистичный или более игровой.
2. Перенести прототип в `Next.js`.
3. Подключить `Supabase`.
4. Сделать приватный вход для двух пользователей.
5. Переделать Виселицу из локальной игры в игру с комнатой и realtime.
6. Добавить первую настоящую галерею с загрузкой файлов.
